# Chapter1

## 
我以前看一个陌生女人的来信，觉得不太接受。但是写到一半的时候回头看，可能这也像是“一封陌生男子的来信”。但是男生和女生的生活，毕竟是不一样的。但是，它甚至无法用“一封陌生男子的来信”这样暧昧不清的词语来粉饰，显得好像一往情深的样子一样。毕竟一个人去纠缠他的前任，且不说他有没有出息，纠缠许多年并不能证明它多么爱前任，说不定就只是他过得不好而已。真是奇妙，茨威格的小说大家都喜欢，可是当我摆出一副一往情深的姿态，我显得这么自私。

我大概平均一年会侧面打听一次你后来是什么样的生活，你且先不要想这些文字是否适宜送到你这里。这实在是一封不足为外人道、也未必适宜的信，我并不能证明它为什么值得你看，但我会觉得它不是不值得看的，毕竟是我自己写的嘛。但是他对我的意义，就是如果说死前想把什么事情做完的话，就只有把它写完吧。我想死之前能看看这东西，不管是老死的还是自杀的。

## 
正如之前说的，我来德国的那一年，有一天我听到宋东野唱，“昨天我数到第二十五颗星星，在北京第二十五个秋天的夜晚”。那时候你快要过二十五岁的生日了，我心里面动了写点东西的念头。拖了这么多年，还是没有写好一个东西，我曾经设想过这个东西它很简短，很直接，但是它拖了许多年；我以为我能写的很恶心，很暴躁，可是它只像某种无能狂怒；我设想过它会很宏大，很曲折，我做过一个设想，希望有机会能把他做成一个幽微曲折、剧情复杂的网页游戏，可是我创作不出来；我觉得它应该更有趣，更复杂，更值得一看，可是能力有限，我确实没有做到。

## 

但是还是让我尽量维持一种轻松的气氛，再来讲一个故事吧。不知道是否你曾记得我一万年以前有一次在德国疯疯癫癫地告诉你，我过两天想回国见见你，去北京待一会儿就回德国。可是我终究也没有回去，2017年3月27号我离开亲爱的祖国，直到2019年4月1号才第一次回来。回来之后我找了附近的小店买了一张电话卡，然后就打车去了中航科技产业园，长城华冠距离机场真的好近啊，打个车十几分钟就到了。

在门口犹豫了一会儿要不要给你打电话，犹豫来犹豫去，看了一会儿许多浩浩荡荡的队伍从那个大门走进走出，还是鼓足勇气用刚买的电话卡给你打了个电话。结果打过去之后你那边却毫无反应。。。。我跟我同学正聊微信来着，我跟她说没有反应，微信里面两个人是一阵一阵的“哈哈哈哈哈哈哈哈哈哈”，聊天从我紧张的情绪就变成幽默的气氛了。

在中航国际产业园那里，有一辆公交车叫空港五路，到机场特别快，我当时候看到那个空港五路那个名字，突然想起来宋东野的《空港曲》。“但愿不在此时此地，但愿谁都不在意。”

结果在回山东的路上，电话打回来了。那时候我觉得真是很有意思了，我跟我朋友用更长的“哈哈哈哈哈哈哈哈哈”来表示更想笑了。我在回去的路上看看刚回来满身狼狈的自己，怂怂地没有接这通电话。

我之前的时候说从你们公司打车到机场好近，何止近，坐公交车也只要一会儿就到了。我也是有很多小情绪的呀。因为走的时候终归是没有能见你一下，虽然这么近。你看，虽然我想尽量写得诙谐一点，但总归还是有很多小孩子的情绪吧，有一些小小的戾气在里面。不过这要比刚写的时候柔和的多了已经。



## 
就像之前说的，第一次动这个念头，你二十五岁生日前夕那时候。我想起来，送你个什么呢，送你个自由吧。我消失掉，不打扰你了。

自由啊，挺有意思的词儿。

多年以后，我又听到了五月天在唱，“如果有，就让你自由，让你自由~ ~ ~ ~ ~”。

我现在还是记得第一次听这首歌是有一天在仙林那边，你还记得南京这个地理区域吗。有一天晚上我们在吵架，谁也不理谁，走在那边，走来走去谁也不说话。走到广场上，周围在放这首歌，突然我们就拥抱到了一起。我记得这首歌，五月天的《温柔》。“不知道不明了不想要，为什么，明明想要靠近，却孤单到黎明”，我记得就这一句印象好深啊，小姑娘，就记得那个晚上，五月天在唱这句话，反复地唱，我们那时候就突然拥抱到了一起。我好怀念那个时候啊，我已经记不起是哪里的广场了，金鹰广场，或者不在仙林在森林摩尔？你也不记得了吧。我只记得突然拥抱的时候那个感觉如此深刻，我们无数次吵架，无数次讨厌，无数次冷战，无数次谁也不理睬谁，也许多次突然就用抱在了一起，我们突然就和好了。

我们总是这样，漫长的冷战，突然地、炽烈地和好，然后又是漫长的冷战、吵架、闹分手，在下马坊的地铁站前，在三号路上，在操场上，在校风碑那里，在你高中朋友拜访你的时候。我让你失望了，我给你丢脸了，，我们不愉快了，可是又突然，我们又忽然抱在一起了。

## 
睡不着的时候，等公交车的时候，抽烟发呆的时候，想起一点什么来就在这里写下来一两句，有空的时候就删删重复的。我想起来的过去，好像总是我们在闹分手。你总是想分手，我也总是闹分手，我们总是吵架，谁也不理谁，我们有时候很恩爱，很开心。这里对我来说，像是某种地层、年轮一样的东西。一层层的堆积，一点点的变化，有时候豪情万丈，有时候阴郁绝望，一会儿哭，一会儿笑，一会儿愤怒，一会儿绝望。

哦，对了，分手几年之后，我也开始抽烟了，这倒不是因为分手的原因，不过确实是生活有压力，我开始抽烟发呆来缓解压力了。虽然很滑稽，抽烟的时候经常也并不过肺，也并没有那么有感觉，只是在发呆，在排解而已。

一开头，我的朋友，你的朋友，你的朋友胡先生几年前记下来了，早上在回到过去的梦里面醒过来这件事情，这样“从别后，忆相逢”的荒诞故事，后来成了他生活中的保留节目。

可是我并不是想把我渲染的多么痴情的样子。你的朋友，他，他虽说时常挂念着前任的，可是也并没有三贞九烈玉洁冰清，他到底是分手之后也追过别人，那时节他跟那个女生，他们两个人手捧着咖啡又眼望着远方，夕阳西下也很好看，这浪漫的时节，你的朋友略一沉吟，转过头来跟她说话，应该是要说什么浪漫的话吧，可是他跟她却聊起来，记挂前任、怀金悼玉、睹物思人真是念念不忘怎么就是不见必有回响。每次想起来这个荒诞不经的故事我都觉得我无法想象对面的女孩子得要用怎样一种关爱智障的眼神看着他。

后来我有一次在公园玩，坐在草坪上很惬意很放松，就讲了起来。。。我有一个朋友，曾经他真是很幼稚，他一直喋喋不休的跟女生说他真的好喜欢上一任呢。

坐在对面的人笑盈盈的就看着我说，这个人就是你吧？

我一直觉得，这个“这个朋友就是你的故事”更加适合做一个开头，因为这是我想给你看的，并不是什么那么痴情的姿态，而是一种，有一些浑沌的、矛盾的，想放下又不要放下的状态。
（3）
好像你没有我的微博，那一年我还在微博上发了一张你的照片，配了那一句“昨天我数到，二十五颗星星，在北京，第二十五颗星星的夜晚”，感慨你要二十五岁了。我刚刚还写过，我有追过一个女生来着，我还有一次跟那个女生喝咖啡的时候不知道为什么就给她看了这条微博，我也不知道为什么，虽说事后我就只觉得自己是一个神经病一样。但是我还是在一直做这样子的事情。可是我还是在做这样的事情，我会把之前对面的的人关爱智障的微表情忘得一干二净然后重新体验一次尴尬。

我不知道你对我的印象还剩下多少，我总觉得你会很快忘了我，从最开始的一些细节开始，到我的外貌，最后我们的种种故事，我总觉得你很快都会把我彻底忘了。

只是倘若你还对我有一点点的印象，我觉得你一定会觉得这样的事情在这样的我身上发生，是一件合理的事情。

我不知道这个题目，你是不是会觉得很俗气，可是我实在是没想到更合适的，这是陆游的《沈园二首》，比起来“红酥手，黄縢酒，满城春色宫墙柳”那一首，我总觉得陆游四十年之后写的这一句“曾是惊鸿照影来”更加的有共鸣，没有那么那么矫情那么“艳”。

你记不记得，那一年你跟我说你要跟别的人在一起的时候我给你写的，“柳花夙有何冤业，萍末相遭乃尔奇”，“直到化泥方是聚，只今堕水未成离”。我很喜欢那一首因为我觉得“萍末相遭”合了萍字和远字，我一直相信你跟我在某一个维度上还没有分离。

我一直在挣扎着让自己相信这件事，我不想放弃。

但我其实也没有想的那么惨了，也不是说你一直是我生活生活的主旋律，只是说像之前说的，你一直存在在我的生活里面，以一种影影绰绰的样子存在着。

曾经我觉得那一段过去，是一件不能忘记的事情，是一个那么重要的存在，一年一年的过去，我心里面给自己许下的许多滥俗的辞藻，也渐渐的袪魅。于是我把那样子的话删了一些，不然你看到了，肯定会心里面觉得尴尬吧。

许多事情，那时候我无奈不已，与为什么我们要分开的任何一件小事情，都让我激动不已。我临走的时候跟你说话，总是歇斯底里的，不知道你还记不记得。只有这许多年过去之后，我这个幼稚的孩子才终于明白，我不应该这个样子，可是即使我现在懂了，如果让我再穿越回到从前，我还是觉得我不能控制自己的情绪。只是说我之前说的，“袪魅”之后，我能明白我所陶醉和无法自拔的情绪，并不是什么崇高和深刻的“爱情”，我歇斯底里，可能一大半就是因为我只是歇斯底里。

（6）
我真的是喜欢极了我走的时候你发的那一句，

	“从我与他相熟，我就知道他的内心住着一个少年，
	他是我的少年”。

我喜欢这句话，我爱极了这句话，我觉得我为这句话去死也死而无憾了。

我原来很开心，我在你这里还是有一些东西留了下来，有那样一些影响，是你总也抹不掉的。可是后来好像关于我的存在，你越来越来多的删掉了。梅香街十号，这不是我起的名字吗，怎么一看，不是这个名字了呢。怎么名字换了呢，不是说好的搬家了也不改名吗？

我说我看到你还是在放热河，心里一股无名火起，我说你看到你还转发了倪一宁的微博，心里面一阵冷笑，你对这些会觉得很不爽觉得很无语的对吧。你最后一篇更新关于武林外传，我心里面觉得写的不太行，很不太行，感觉跟以前知乎上好多人用这个套子也在写。你不用生气，我自己也是苦笑一声，我还是没有改啊，我这臭毛病，真的是个臭毛病呢。

你记不记得你有一次发了公众号我特别歇斯底里的给你打过电话去，发了一通脾气，说你不要在我的圈子里做这样的事情。现在我不会了，我看你发的东西，我也不会再激动，不会再跟你说话，也没有办法跟你说话了。
还是说回来吧，时间，时间，时间过得好快啊，你还在听热河吗，你还在想李志吗，你还记得他吗，他被封杀也已经过去了一年了。

我记得我跟你难过了一番，你感慨了两句“往者不可谏，来者要去追”，我还为此大为光火，你是不撞南墙不回头的人，你现在会觉得幸福吗，会觉得比我好吗？会的吧。

我记得你说你那一位答应你之后带你去厦门玩儿，厦门好玩儿吗？我也去了厦门，厦门确实很漂亮。我们各奔东西以后我也去了许多地方，乌镇的乌篷船头，周庄的水乡，澳门的许多粤式地名我不会念。科隆的大教堂很好看，柏林很穷很性感，马德里的街道很宽阔，新宿的夜生活很有趣，涩谷的衣服很酷，京都的枫叶特别好看，布鲁塞尔的啤酒特别好喝。我出国的时候跟自己说，“休对故人思故国，且将新火试新茶”，可是基本没有做到，我走到哪里也会想起你来。你记不记得有一年冬天我突然给你打电话说要给你寄明信片你当场挂了电话，那时候我在布鲁塞尔的撒尿小童旁边来着，哎，现在想想那时候我真冒昧啊。

我觉得我很努力，很多缺点都改了，有一次跟我同学聊天，他说不是的，我其实没有那么努力，不然不可能体型都控制不了。可能是的吧，许多缺点我都没有变，这些年好像离开你之后反而变本加厉了。

你很了解我吧，你深知我和你的不合适，可是你还是包容了我三年。我很让你失望吧，我是那么的不懂事。或许你也曾经鼓起勇气要爱我，可是我总是五分钟之后就会让你失望，可能我终究不是你值得爱的人。“男孩子的幼稚和女孩子的悲观，就是爱情的悲剧”，我常会想起来。我总是满嘴毒鸡汤，但是好像并不能做到什么。

许多次我觉得可能我不要结婚了，不要谈恋爱了，像我这样的人，始终就像一个只会读书的书呆子把，谢谢你让我看到了许多我从来没有想过的看过的东西。我记得后来认识许多人，许多事情他们都会说，没想到你抽烟，没想到你这么喜欢喝酒。他们总是说，你看起来就像一个乖小孩。乖小孩，这话反反复复的在出现。即使我讨厌我始终是这个样子，可是我好像也不太能改变的了我了。
